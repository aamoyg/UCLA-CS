1-1.    (a) A cell is a word in memory which can hold an instruction or data.
(p9)

    (b) Two's complement is a method of representing negative numbers in which
all digits are complemented, one unit is added to the least significant digit
position, and any carry from the operation is discarded. (p10)

    (c) The American Standard Code for Information Interchange (ASCII) is a 7-
bit code for representing alphnumeric character data. (p13)

    (d) Overflow is a condition which occurs when the result of an arithmetic
operation requires more digits to represent the number than are available in
the cell which is to contain the result. (p15)

    (e) A mantissa is the fractional part of a floating-point number which is
multiplied by the base raised to the exponent to obtain the value of the
number. (p21)

    (f) A normalized number is a number in scientific notation which conforms
to a format which has exactly one digit before the radix point and is
multiplied by a power of the radix to obtain the actual value. (p22)

1-12.   191744(10)  =   101110110100000000(2)
            1728(10)    =   110110000000(2)

The binary point must be moved 17 places to the left.
There must be 17 bits plus 1 sign bit.
Thus use 18 bits for the mantissa and 6 bits for the exponent.
This allows exponents in the range -32 to +31 which is sufficient.
